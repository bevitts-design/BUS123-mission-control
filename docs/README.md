# BUS123 Course Operating System Documentation

This folder contains the working documentation for the BUS123 Course Operating System.

## Core Files

- `01_System_Architecture.md` — overview of repositories, data sources, publishing targets, and workflows.
- `02_Master_Lesson_Inventory.md` — human-readable lesson inventory generated from the canonical lesson data.

## Data Files

The canonical data files will live outside this folder in `/data`:

- `data/BUS123_CANONICAL_LESSON_INVENTORY.csv`
- `data/BUS123_CANONICAL_LESSON_INVENTORY.json`

## Operating Principle

Edit data once. Publish everywhere.

The lesson inventory should become the source of truth for the student website, Canvas workflows, QTI generation, and validation tools.
